# Промт для GPT: Колобок, слои персонажа

Сгенерируй набор ассетов для 2D-скелетной анимации персонажа детского приложения для чтения. Персонаж называется **Колобок**. Стиль: добрый, мягкий, 3D-cartoon, как оранжевый круглый “боб” из детской игры, тёплые цвета, крупные милые глаза, мягкие тени, без страшных деталей. Возраст аудитории 6–7 лет.

Важное требование: **каждый ассет отдельным изображением**, не коллажем. Если прозрачный фон недоступен, сделай однотонный белый или ярко-зелёный фон без теней по краям объекта, чтобы фон можно было легко удалить онлайн-редактором.

Общие правила для всех изображений:

- одинаковый стиль, свет и ракурс;
- объект по центру;
- без текста, подписей, рамок, логотипов;
- без лишних предметов;
- высокий контраст на краях;
- формат предпочтительно PNG/WebP;
- объект целиком помещается в кадр с небольшим запасом;
- размер 1024×1024 для частей персонажа, 1536×1024 для фона.

## Эволюция, 6 стадий

Сделай 6 стадий одного персонажа. Колобок на 6-й стадии — взрослая особь. Стадии должны читаться как развитие:

1. **Головастик / яйцо**: маленькая оранжевая капля с мягким хвостиком, без рук, очень простой силуэт.
2. **Росток**: круглее, хвост меньше, намёк на маленькие боковые выступы.
3. **Малыш**: маленький круглый Колобок с короткими ручками-каплями.
4. **Подросток**: более уверенная форма, ручки заметнее, корпус чуть крупнее.
5. **Юный Колобок**: почти взрослый, аккуратная круглая форма, дружелюбный вид.
6. **Взрослый Колобок**: финальный милый оранжевый боб, круглый, мягкий, с маленькими руками.

## Общие эффекты

Дополнительно сгенерируй:

```text
effects/star.png
effects/sparkle.png
effects/comet.png
```

Требования:

- `star.png`: мягкая жёлтая пятиконечная звезда с свечением, без фона.
- `sparkle.png`: маленькая искра.
- `comet.png`: звезда со световым хвостом для полёта вокруг персонажа.

## Анимации, под которые нужны слои

Слои будут использоваться во Flutter:

- **idle**: лёгкое дыхание, масштаб 1.00 → 1.04 → 1.00.
- **jump**: тело подпрыгивает, тень сжимается.
- **spin**: персонаж быстро крутится вокруг вертикальной оси или делает весёлый поворот.
- **wink**: `eyes_open` меняется на `eyes_wink` на 150–220 мс.
- **joy**: персонаж радуется, вокруг летит `star`/`comet`.

Особенно важно: глаза должны совпадать по положению с телом на всех стадиях. Если фон не прозрачный, сделай фон одинаковым однотонным, чтобы удалить его пакетно.
