# Закладка-окно

## Чем занят ребёнок

На экране виден **фрагмент текста** — одна строка или одно слово — через **«окошко»**. Окно **быстро перемещается** по тексту. Ребёнок читает только то, что внутри рамки.

## Зачем это нужно

В книге ребёнка часто отвлекает **всё сразу**: картинки, соседние строки, конец абзаца. Закладка-окно:

- **сужает поле внимания** до одного фрагмента;
- тренирует **последовательное продвижение** по тексту;
- снижает тревогу «я не успею всё прочитать».

Это промежуточный шаг между **одним словом** (тахистоскоп, RSVP) и **целой страницей** учебника.

## Связь с другими упражнениями

| Этап | Упражнение |
|------|------------|
| Узнать слово целиком | Тахистоскоп |
| Держать ритм | RSVP |
| Пройти строку без отвлечений | Закладка-окно |
| Охватить страницу | Книга / комикс в приложении (позже) |

## Как поддержать дома

- Читайте вслух **вместе** первые проходы — ребёнок сверяет свой темп с вашим.
- Увеличивайте длину фрагмента **постепенно**: слово → короткая фраза → строка.
