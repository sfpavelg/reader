# Для мамочек

Этот раздел — для вас. Здесь простым языком объясняется, **чем занят ребёнок** в каждом упражнении и **зачем это нужно** для развития чтения в 6–7 лет.

## Почему это приложение вообще важно

В 6–7 лет дети обычно уже знают буквы и читают по слогам. Главная задача этого возраста — **перейти от слогового чтения к целостному восприятию слова** и сделать чтение **автоматическим**, без «застревания» на каждой букве.

Наше приложение тренирует не «скорость ради скорости», а фундамент, без которого быстрое чтение невозможно:

- **зрительную память** — ребёнок узнаёт слово целиком, а не собирает его заново;
- **ширину захвата** — глаз видит больше букв за один взгляд;
- **скорость распознавания** — меньше пауз между словами;
- **устойчивое внимание** — без отвлечений и регрессии (возврата к прочитанному).

Все упражнения **офлайн**, без рекламы в момент тренировки, с **короткими сессиями** (рекомендуем 10–15 минут в день) — это бережно к глазам и нервной системе.

## Мотивация для ребёнка

Ребёнок 6–7 лет редко понимает абстрактную пользу «быстрого чтения». Поэтому в приложении есть [игровые механики](gamification.md): питомец-читатель, звёзды, альбом наклеек, карта миров. Это **мягкая мотивация** — без таймеров «на провал» и без наказания за ошибку.
