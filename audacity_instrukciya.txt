AUDACITY — ИНСТРУКЦИЯ ДЛЯ ОЗВУЧКИ СКАЗОК (Обучайка / Reader)
Версия для Audacity 3.7.x (Windows 64-bit)
Сохраните этот файл рядом с audacity-win-3.7.8-64bit.exe
================================================================================

ЧТО ЭТО
--------
Audacity — бесплатная программа для записи, правки и экспорта звука.
Официальный сайт: https://www.audacityteam.org/download/
Нужен файл: audacity-win-3.7.8-64bit.exe (прямой установщик).
Не нужен: MuseHub и платные «bundle deal / tutorial».


УСТАНОВКА НА ДРУГОМ ПК
----------------------
1. Скопируйте на ПК:
   - audacity-win-3.7.8-64bit.exe
   - этот файл инструкции (.txt)
2. Запустите установщик.
3. Если Windows спросит SmartScreen / «Бег» / «Не запускать» —
   для официального файла можно нажать «Бег» (Запустить).
4. Установка: «Далее» до конца.
5. При первом запуске:
   - «Принять и продолжить» (обновления/статистика)
   - Рекламу Audacity 4 beta — «OK», лучше с галочкой
     «больше не показывать». Бета НЕ нужна.


================================================================================
1. КАК ПИСАТЬ (ЗАПИСЬ)
================================================================================

ПОДГОТОВКА
----------
- Тихая комната, микрофон ближе к рту (~15–25 см), без эха.
- Закройте шумные программы, отключите уведомления.
- Говорите чуть медленнее обычной речи (для детей 6–7 лет).
- Текст держите перед глазами; делайте короткие паузы между фразами.

ВЫБОР МИКРОФОНА
---------------
1. Откройте Audacity.
2. Вверху на панели устройств выберите ваш микрофон
   (не «стерео микшер» и не динамики, если пишете голос).
3. Каналы: лучше «1 (Mono)» — моно, файл будет меньше.

ПРОВЕРКА УРОВНЯ
----------------
1. Говорите пробную фразу и смотрите на полоски уровня.
2. Громкость должна быть заметной, но без «зашкала» в красную зону.
3. Если тихо — ближе к микрофону или громче в настройках Windows
   (Параметры → Система → Звук → Ввод).

ЗАПИСЬ
------
1. Нажмите красную кнопку записи (●).
2. Подождите 1 секунду тишины, затем читайте.
3. В конце — 1 секунда тишины.
4. Нажмите Stop (жёлтый квадрат / Стоп).
5. Послушайте: зелёная кнопка Play (▶).

СОВЕТ ПО ДЛИНЕ
--------------
Не пишите сразу всю главу.
Лучше короткие куски по 1–3 минуты:
  01_intro.ogg
  02_hurricane.ogg
Так проще перезаписать и заменить один кусок.


================================================================================
2. КАК РЕДАКТИРОВАТЬ
================================================================================

ВЫДЕЛЕНИЕ
---------
Инструмент «Выделение» (I) — протяните мышью по дорожке.

ОБРЕЗАТЬ ТИШИНУ В НАЧАЛЕ/КОНЦЕ
-----------------------------
1. Выделите тишину в начале.
2. Delete / Del.
3. То же в конце.

УБРАТЬ ОШИБКУ / КАШЕЛЬ / ДОЛГУЮ ПАУЗУ
------------------------------------
1. Выделите плохой кусок.
2. Delete.
3. Если нужно вставить паузу: Выделить место →
   меню «Генерировать» → «Тишина…» (0.3–0.8 сек).

СДЕЛАТЬ ГРОМЧЕ / ТИШЕ
---------------------
1. Выделить весь трек: Ctrl+A.
2. Меню «Эффекты» → «Громкость и сжатие» → «Нормализация…»
   (или «Усиление…»).
3. Нормализация обычно удобнее: выравнивает громкость.

УБРАТЬ ШУМ (ШУРШАНИЕ КОМПЬЮТЕРА)
--------------------------------
1. Найдите кусок ТОЛЬКО с шумом (без голоса), 0.5–1 сек.
2. Выделите его.
3. «Эффекты» → «Шум и восстановление» → «Удаление шума…»
   → «Получить профиль шума».
4. Выделите весь трек (Ctrl+A).
5. Снова «Удаление шума…» → ОКменить (не переусердствуйте,
   голос не должен стать «роботом»).

СКЛЕИТЬ ДВА КУСКА
-----------------
1. Запишите второй кусок (появится новый трек) ИЛИ
   импортируйте второй файл: «Файл» → «Импортировать» → «Аудио…».
2. Подгоните куски по времени (инструмент «Смещение по времени»).
3. При экспорте Audacity сведёт в один файл.

ОТМЕНА ДЕЙСТВИЯ
---------------
Ctrl+Z — отменить.


================================================================================
3. КАК СЖИМАТЬ (УМЕНЬШИТЬ ОБЪЁМ ФАЙЛА)
================================================================================

ГЛАВНОЕ ПРАВИЛО
---------------
Для приложения нужен маленький файл:
- моно (1 канал)
- голос, не музыкальный «hi-fi»
- целевой битрейт примерно 24–48 кбит/с

РЕКОМЕНДУЕМЫЙ ФОРМАТ ДЛЯ ПРИЛОЖЕНИЯ
----------------------------------
Лучше всего: OGG (Vorbis) или Opus, если есть в списке.
Запасной вариант: MP3.

ЭКСПОРТ В AUDACITY (ОКНО «Экспорт аудиоданных»)
-----------------------------------------------
Путь: «Файл» → «Экспортировать» → «Экспортировать аудио…»

РЕКОМЕНДУЕМЫЕ НАСТРОЙКИ ДЛЯ ПРИЛОЖЕНИЯ (голос / сказки)
------------------------------------------------------
Имя файла:          havroshechka.ogg  (латиница, без пробелов)
Папка:              любая удобная (например Documents\Audacity),
                    потом скопируйте файл в проект Reader
Формат:             Ogg Vorbis

Параметры аудио:
  Каналы:                 Моно
  Частота дискретизации:  44100 Hz
  Качество:               2 … 4   (ползунок 0–10)

Диапазон экспорта:
  Весь проект
  «Обрезать пустое пространство перед первым клипом» —
      можно включить (галочка), если в начале дорожки тишина

Кнопка: «Экспорт»

ЧТО ЗНАЧАТ ПОЛЯ
---------------
Имя файла     — как файл сохранится на диске.
Папка         — куда сохранить; «Обзор…» меняет путь.
Формат        — тип файла. Для приложения: Ogg Vorbis
                (запасной вариант: MP3).
Каналы        — Моно: один канал, файл меньше, для голоса ок.
Частота       — 44100 Hz стандарт; не ставьте выше без нужды.
Качество      — чем меньше число, тем сильнее сжатие и меньше
                размер, но голос хуже.
                  0–1  — очень маленький файл, голос «хрипит»
                  2–4  — хороший баланс для сказок (рекомендуем)
                  5–7  — чище, файл больше
                  8–10 — почти без экономии размера
Весь проект   — экспортировать всю дорожку целиком.
Метаданные    — «Правка метаданных…» можно не трогать.

ПРИМЕР ХОРОШЕЙ КОНФИГУРАЦИИ
---------------------------
  Формат:     Ogg Vorbis
  Каналы:     Моно
  Частота:    44100 Hz
  Качество:   3
  Диапазон:   Весь проект

ПРИМЕЧАНИЕ
----------
Качество «1» сильно экономит место, но для детской озвучки
лучше сначала послушать результат. Если голос «булькает» —
поднимите до 3 или 4 и экспортируйте снова.

ПРИМЕРНЫЙ РАЗМЕР
----------------
1 минута речи ≈ 0.2–0.5 МБ при сильном сжатии.
3 минуты ≈ 0.6–1.5 МБ.
Целая глава 10 минут ≈ 2–4 МБ (зависит от настроек).

ЕСЛИ НУЖНО ЕЩЁ СИЛЬНЕЕ СЖАТЬ
----------------------------
После экспорта можно дополнительно прогнать через FFmpeg
(для продвинутых). Для старта достаточно экспорта из Audacity.

НЕ ИСПОЛЬЗУЙТЕ ДЛЯ ПРИЛОЖЕНИЯ
----------------------------
- WAV / FLAC — очень большие файлы
- стерео без необходимости — почти двойной объём


================================================================================
4. ЧТО ЕЩЁ ВАЖНО ЗНАТЬ
================================================================================

СТРУКТУРА ФАЙЛОВ ДЛЯ ПРИЛОЖЕНИЯ READER
-------------------------------------
Кладём так (имена важны):

assets/audio/fairytales/
  oz/
    oz_1/
      01_trial.ogg
      02_....ogg
    oz_2/
      ...
  hansel/
    ...

Где:
- oz     = id сказки («Волшебник изумрудного города»)
- oz_1   = id главы («Ураган»)
- 01_... = короткий сегмент главы

После записи положите файл в проект и напишите ассистенту «готово».

ВАЖНО ДЛЯ FLUTTER (подпапки audio)
---------------------------------
В pubspec.yaml нужно указывать папку СКАЗКИ, а не только fairytales/:
  - assets/audio/fairytales/havroshechka/
Иначе файл в подпапке может не попасть в сборку приложения.

КОРОТКИЕ СЕГМЕНТЫ ЛУЧШЕ ОДНОЙ ДЛИННОЙ ДОРОЖКИ
---------------------------------------------
Плюсы:
- легче перезаписать один кусок
- меньше риск «испортить всё»
- проще потом заменить/докачать в приложении

ПРОЕКТ AUDACITY (.aup3)
-----------------------
«Файл» → «Сохранить проект» — сохраняет исходник для правок.
В приложение кладём НЕ проект, а ЭКСПОРТ (ogg/mp3).

РЕЗЕРВНАЯ КОПИЯ
---------------
Храните:
1) исходный проект Audacity
2) финальный сжатый ogg/mp3
На случай, если понадобится переэкспорт.

ЮРИДИЧЕСКИ (КРАТКО)
-------------------
Свой голос ≠ автоматическое право публиковать чужой текст.
- Дома / тест / MVP — обычно ок.
- Публикация в магазине приложений полной озвучки
  авторских книг (Волков, Толстой и т.п.) может требовать прав
  или своих пересказов / текстов общественного достояния.
Перед релизом в Google Play лучше уточнить по конкретным сказкам.

ЕСЛИ НЕТ ЗВУКА ПРИ ЗАПИСИ
------------------------
1. Проверьте микрофон в Windows (Параметры → Звук → Ввод).
2. В Audacity выбран правильный микрофон.
3. Не выбран режим «стерео микшер» вместо микрофона.
4. Разрешите Audacity доступ к микрофону в Windows.

ЕСЛИ ФАЙЛ НЕ ЭКСПОРТИРУЕТСЯ В MP3
--------------------------------
В новых Audacity MP3 обычно есть.
Если нет — экспортируйте в OGG (для приложения даже предпочтительнее)
или поставьте поддержку FFmpeg по подсказке Audacity.

КОНТАКТ ДЛЯ ИНТЕГРАЦИИ В ПРИЛОЖЕНИЕ
-----------------------------------

«готово, файл: assets/audio/fairytales/oz/oz_1/01_trial.ogg»
